## About 👩🏻‍💻 <a id="about"></a>
This assignment was created to demonstrate OOP principles in PHP, including class hierarchy, inheritance, interfaces, and traits.

## Hierarchy 🌿 <a id="hierarchy"></a>
- **Food** — root parent class
  - **Fruit** — extends Food
    - **Apple** — extends Fruit
    - **Banana** — extends Fruit
  - **Grain** — extends Food
    - **Rice** — extends Grain
  - **Protein** — extends Food
    - **Chicken** — extends Protein

## Features ✨ <a id="features"></a>
  ⚡️ Three levels of class hierarchy<br/>
  ⚡️ Interface defined alongside root class<br/>
  ⚡️ NutritionTrait shared between Fruit and Grain<br/>
  ⚡️ spl_autoload_register for class autoloading<br/>
  ⚡️ PHP Doc blocks and type hints throughout<br/>

## Installation 📦 <a id="installation"></a>
To run locally:
- Clone this repository
- Set up a local server (WAMP or MAMP)
- Move the project folder into `C:/wamp64/www/` (WAMP) or `/Applications/MAMP/htdocs/` (MAMP)
- Navigate to `http://localhost/oop-assignment/` in your browser to view
